# 📌 Project Overview

This project is a browser-based Tic Tac Toe game that enables two-player interaction with a simple and intuitive interface. Players can choose their symbol before starting the game, and the system dynamically manages turns, detects winners, and displays results.

# 🚀 Features
- Player selection (X or O)
- Interactive 3x3 game board
- Real-time turn indicator
- Winner detection logic
- Replay functionality
- Clean and responsive UI

# 🛠️ Technologies Used
- HTML5 – Structure of the game
- CSS3 – Styling and layout design
- JavaScript (Vanilla JS) – Game logic and interactivity

# 📂 Project Structure

📁 Tic-Tac-Toe<br>
│── index.html<br>

# ⚙️ How It Works
- User selects Player X or Player O
- Game board initializes
- Players take turns automatically
- JavaScript checks winning combinations
- Result is displayed when a player wins or game draws
- Replay button resets the game
